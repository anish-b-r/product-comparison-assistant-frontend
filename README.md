# 🎨 AI Visual Price Comparison Assistant - Frontend Web UI

A premium, modern web dashboard built using Next.js, React, and Tailwind CSS. It serves as the frontend interface for the AI Visual Price Comparison Assistant, providing users with a beautiful, intuitive platform to search for products, view price ranges across major online stores, analyze savings, and read tailored shopping suggestions powered by Google Gemini 2.5.

---

## 🌟 Key Features

* **Ambient Glassmorphic UI**: High-fidelity dark-mode-influenced light UI with decorative glowing backgrounds, smooth transitions, and responsive hover transformations.
* **Price Analytics Grid**: Highlight cards for:
  - **Lowest Price**: Shows the minimum pricing found along with the platform name.
  - **Highest Price**: Lists the maximum market price found.
  - **Maximum Savings**: Quantifies exactly how much money is saved (both absolute value in ₹ and percentage) by choosing the cheapest store.
* **Price Spectrum Visualization**: A dynamic comparison bar chart showing relative deals. It scores offers relative to the lowest price and highlights the pricing percentage increase (e.g. `+12%`).
* **Platform Features Matrix**: A detailed tabular matrix presenting:
  - Custom brand icons and e-commerce platforms.
  - Interactive badges showing shipping details (e.g. "Free Delivery").
  - Verification badges (e.g. "Verified Store") representing trusted domains.
  - Outbound links to purchase directly from the store.
* **Side-by-Side Offer Cards**: Individual cards for Amazon and Flipkart displaying price points and quick navigation links.
* **Gemini Recommendation Panel**: A curated card summarizing purchase advice dynamically generated by the Gemini model.
* **Dynamic Sidebar**:
  - **Search History List**: Lists past searches, which can be clicked to reload comparison data instantly.
  - **Delete Specific Item**: Hover over any history item to delete it via a trash icon.
  - **Clear All**: Bulk deletes all history records with an overlay confirmation safety modal.
* **Quick Search suggestions**: Ready-made search pills for devices like "iPhone 17", "Samsung S25", and "Sony WH-1000XM5" to easily test the application's flow.
* **Progress Loading & Error Alerts**: Real-time loading states showing backend pipeline progress (e.g., "Searching e-commerce platforms...") and visual alert notifications if operations fail.

## 🛠️ Technology Stack

* **Client Framework**: [Next.js 15](https://nextjs.org/) (App Router, Client-Side Rendering with `"use client"`)
* **Core Logic**: [React 19](https://react.dev/) - hooks (`useState`, `useEffect`, `useRef`) for component state management
* **Language**: [TypeScript](https://www.typescriptlang.org/) - strict type definitions for search payloads and schemas
* **Styling & Theme**: [Tailwind CSS](https://tailwindcss.com/) - utility classes for custom styling, layouts, and animations
* **Icons**: [Lucide React](https://lucide.dev/) - clean, vector-based line icons

## ⚙️ Configuration

The frontend communicates with the FastAPI backend. You can find the backend URL configured at:

* **File**: `frontend/src/app/page.tsx`
* **Line**: `const BACKEND_URL = "http://localhost:8000/api";`

Ensure the backend server is running on `http://localhost:8000` (or update this variable to point to your deployed backend URL).

## 🚀 Installation & Running Locally

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed (v18.x or higher is recommended).

### 2. Install Node Packages
Navigate to the `frontend/` folder and install packages:
```bash
cd frontend
npm install
```

### 3. Run the Development Server
Start the Next.js development server:
```bash
npm run dev
```
Open **`http://localhost:3000`** in your browser to view the application.

### 4. Build for Production
To package the app for production:
```bash
# Build the optimized production bundle
npm run build

# Start the server in production mode
npm start
```
